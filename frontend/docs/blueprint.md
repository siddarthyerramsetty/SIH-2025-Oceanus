# **App Name**: Oceanus Insights

## Core Features:

- Dynamic Map Visualization: Interactive map of the Indian Ocean using Leaflet.js, displaying real-time data from Argo floats.
- Animated Ocean Layer: Background ocean layer with a continuous, textured water flow animation, simulating ocean currents using Canvas or WebGL overlay on Leaflet.
- Argo Float Visualization: Visualize Argo floats as pulsating, glowing dots on the map, with pop-up metadata on click.
- Dual-Mode Chatbot: Chatbot with education and scientist modes for querying Argo data.
- Educational Response Generation: In education mode, the chatbot generates simple, easy-to-understand responses with proactively suggested example questions to guide the user.
- Context Aware Tool Use: In scientist mode, the chatbot will respond in a detailed manner appropriate for experts. It uses a tool for the incorporation of external Argo data to support its output, as appropriate.
- Initial Guidance Modals: Elegant modals to guide first-time users on map interaction and chatbot usage.

## Style Guidelines:

- Primary color: Light, off-white background (#F8F9FA) for a crisp and clean feel, in the Daylight Nautical theme.
- Secondary color: Light grey panels (#E9ECEF) to provide subtle contrast and separation of content areas, in the Daylight Nautical theme.
- Accent color: Bright royal blue (#007BFF) for highlights and interactive elements, ensuring visibility and engagement, in the Daylight Nautical theme.
- Text color: Very dark grey/almost black (#212529) for high readability against the light background, in the Daylight Nautical theme.
- Body and headline font: 'Inter', a modern sans-serif font for a clean and accessible user experience.
- Simple, professional icons (globe for map, message-circle for chatbot) from a consistent icon set (e.g., Font Awesome or Remix Icon).
- Fixed sidebar on the left for navigation, with the main content area on the right.
- Subtle animations for transitions and interactions (e.g., fade-in effects, smooth scrolling).
